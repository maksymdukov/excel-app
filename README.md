## Features

- zero dependencies
- formula processing (parsing, tokenizing, lexing, interpreter)
- visual cell formatting
- own mini-framework
- resizable cells
- tables can be saved
- redux from scratch


### Patterns used:
Observer, Facade, Interpreter
